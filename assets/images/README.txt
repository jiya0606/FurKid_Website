FURKID RESCUE — IMAGES FOLDER
==============================

This folder holds every photo that appears on your website.

HOW TO ADD OR REPLACE A PHOTO
-----------------------------
1. Save the new photo into this folder (assets/images/).
2. Use a simple filename with NO spaces — for example: pet-rocky.jpg
3. Open content.json and change the matching image path to your new filename.
   (Or ask ChatGPT to do it — see CONTENT-GUIDE.xlsx for the exact prompt.)

RECOMMENDED PHOTO SIZES
-----------------------
- Hero (home page):         1600 x 900 px   (wide, landscape)
- Mission / About story:    1400 x 800 px
- Pet cards:                 800 x 800 px   (square-ish works best)
- Happy tails cards:        1200 x 700 px
- Instagram grid tiles:      600 x 600 px   (square)
- Foster / sponsor sections: 1400 x 800 px

FILE FORMATS
------------
Use .jpg for photos, .png for logos/illustrations, .webp if you know what it is.
Keep files under 400 KB if possible — big files slow down the site.

MISSING A PHOTO?
----------------
If a photo is missing, the site will hide it automatically — no errors. So you
can launch with some photos missing and add them in later.
